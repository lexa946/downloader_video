## Изменения в коде

### Основные изменения в `app/parsers/vk.py`:

1. **Автоматическое определение типа URL**:
   ```python
   self.is_vk_com = "vk.com" in url
   self.is_vkvideo_ru = "vkvideo.ru" in url
   ```

2. **Поддержка разных форматов URL**:
   - `vk.com`: `video335482664_456239334`
   - `vkvideo.ru`: `video-212496568_456248024`

3. **Два метода получения информации**:
   - `_get_vk_com_info()` - для vk.com
   - `_get_vkvideo_info()` - для vkvideo.ru

4. **Унифицированный интерфейс**:
   - Оба метода возвращают одинаковую структуру данных
   - Совместимость с существующим кодом

## Алгоритм работы

### Для vk.com:
1. Извлечение `owner_id` и `video_id` из URL
2. Получение страницы видео для извлечения hash
3. Извлечение названия из meta тега `og:title`
4. Запрос embed страницы с hash
5. Извлечение URL видео из embed страницы
6. Выбор лучшего качества

### Для vkvideo.ru:
1. Извлечение `owner_id` и `video_id` из URL
2. POST запрос к API vkvideo.ru
3. Парсинг JSON ответа
4. Извлечение информации о видео
